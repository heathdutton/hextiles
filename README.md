# Gametile Designer

Interactive tool for designing laser-cut board game tiles that softly interlock via twist-lock joints.

**[Try it live](https://heathdutton.github.io/gametile-designer/)**

![Gametile Designer](screenshot.png)

Supports triangle, square, and hexagon shapes with adjustable twist angle, radius, rounding, and thickness. Preview in 3D, tweak physics gauges, and export production-ready SVG for laser cutting.
